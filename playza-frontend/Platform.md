This platform playza is a game base competitive platform where users enter a game with an entry fee and paly and the leader of the game leaderboard wins the prize pool. The games are played in real-time and the leaderboard is updated in real-time. The platform is built using React, TypeScript, and Node.js. The platform uses WebSockets to communicate between the client and server. The platform uses a PostgreSQL database to store the game data. The platform uses Redis to store the leaderboard data.
