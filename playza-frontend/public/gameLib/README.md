## Project Strategy
We are developing a diverse range of games across five core categories, utilizing **multiple programming languages and engines** to showcase the versatility and performance of the Playza SDK.

### Complete Wave One Feature List:
1. **Neon Runner**: Endless platformer with double jump and parallax environments.
2. **Trivia Master**: High-stakes competitive quiz with timers and streak mechanics.
3. **Crystal Match**: Juicy Match-3 engine with special gems and screen-shaking feedback.
4. **Shadow Strategy League**: Hexagonal turn-based strategy with complex movement and AI combat.

## Integration Guide
Each game in this repository includes a `playza-sdk` integration to handle:
- User Authentication (Session Keys)
- Real-time Score Submission
- Wallet/Reward Logic
- Leaderboards
