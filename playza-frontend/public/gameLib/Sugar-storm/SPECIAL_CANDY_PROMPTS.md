# Sugar Storm — Special Candy Image Prompts

Use these prompts to generate **full special-candy images** (not overlays). Each prompt requests a **background** so you can remove it yourself later.

## Recommended settings

- Format: `1:1`, `512x512` (or `1024x1024` if you want extra crispness)
- Style: casual **match-3 mobile** UI, glossy, vibrant, clean edges, high readability at small sizes
- Background: solid dark navy `#0b1630` with a subtle vignette (easy to cut out)
- Output: PNG/WebP

## Shared negative prompt (use for all)

“text, watermark, logo, UI button, frame, border, realistic photo, blurry, muddy colors, noisy, low detail, artifacts, extra objects, multiple candies, hands”

## Candy palette notes (optional)

- Strawberry: pink/red
- Blueberry: bright blue
- Lemon: yellow
- Mint: green
- Grape: purple
- Orange: orange
